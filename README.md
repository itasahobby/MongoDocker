# MongoDocker

Service to manage a MongoDB docker that automatically creates backups.

## Installation

Use the script to setup the service with root privileges:

```
./setup.sh
```

## Usage

Run service

```
sudo service dockermongodb start
```

Run service

```
sudo service dockermongodb stop
```
